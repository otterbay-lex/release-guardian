# Release Guardian V2

通用型、Agent 主动执行的发布质量守门员。V2 在 V1 的基础检查上，补充工程质量、制品一致性、供应链、中国合规、知识产权和运维恢复能力。

## 与 V1 的关系

- V1 位于 `../v1/`，作为经典基础版独立保留和发布。
- V2 是新增版本，不覆盖或修改 V1。
- 用户可以按需求分别安装两个版本。

## 使用方式

将整个 `v2` 目录安装为 Skill，保留 `SKILL.md`、`references/` 和 `scripts/` 的相对结构。基础扫描器只使用 Python 3 标准库。

## 重要边界

V2 提供基于可访问材料的发布前初步审查，不代替全面安全审计或正式法律意见。检查通过不等于自动获得发布授权。
