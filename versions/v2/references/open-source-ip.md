# 开源与知识产权规则

## 检查对象

检查项目许可证、依赖许可证、复制代码、模板、图片、图标、字体、音视频、模型权重、提示词资源、数据集、文档和品牌素材。

## 判断方法

- 查明来源、作者、许可证文本、版本和分发方式。
- 检查署名、版权声明、源代码提供、相同方式许可、NOTICE 和再分发条件。
- 区分内部使用、网络提供服务、二进制分发和源码公开。
- 不因看到 GPL、AGPL 或其他许可证名称就直接宣布违法；先判断实际组合和分发事实。
- 没有来源或授权证据的重要第三方资源标为高风险；明显无权公开分发时阻断。

项目准备公开开源但缺少许可证时为重要警告；用户明确要授权公众复制、修改和再分发却没有许可证时，发布前应确认许可证选择。
