# 项目画像与专项检查

一个项目可以匹配多个画像。记录识别证据，并合并所有适用规则。

| 画像 | 常见证据 | 追加检查 |
|---|---|---|
| GitHub/开源仓库 | `.git`、远程地址、贡献文件 | 仓库卫生、Issue、SECURITY、贡献规则、许可证 |
| Web/前端 | HTML、前端 manifest、构建配置 | 浏览器密钥、XSS、Cookie、CSP、可访问性、前端制品 |
| API/后端 | 路由、服务器入口、容器配置 | 认证授权、限流、CORS、迁移、日志、健康检查 |
| 桌面/移动 App | 应用 manifest、签名配置 | 权限、签名、更新、本地数据、崩溃恢复、商店披露 |
| CLI/软件包 | CLI 入口、包清单 | 安装卸载、版本兼容、危险命令、注册表发布内容 |
| AI 应用 | 模型 SDK、提示词、对话数据 | 密钥、提示词泄露、数据条款、内容安全、输出提示 |
| Agent Skill | `SKILL.md`、scripts/references | 触发准确性、权限、危险命令、路径可移植、外部依赖 |
| 文档/数据集 | 文档、表格、数据文件 | 来源、隐私、授权、格式、样本真实性、数据说明 |
| ZIP/镜像/制品 | 压缩包、镜像清单、安装包 | 内容清单、哈希、调试残留、秘密、源码一致性 |

无法确定面向地区、用户年龄、是否收费或是否处理真实数据时，先读取代码和文档；仍无法确认且答案会改变结论时，只问一个简短问题。
